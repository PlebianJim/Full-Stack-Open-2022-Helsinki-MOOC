CSS Basics

Cascading Style Sheets (CSS) is a language used in conjunction with HTML that styles content on the page. HTML is used to provide structure, CSS is used to style and JavaScript is used to add functionality and interactivy

CSS controls aspects of styling such as content positioning, colour and size

Taking a closer look at a CSS ruleset

In CSS, you use a selector to select elements, classes and ID's and then assign a style to them. The syntax for CSS is quite simple. The following example shows a selector targeting all <p> elements on a page and changing the colour to green:

p {
    color: green;
}

If you wish to have more fine tuned control over which elements are impacted by the CSS rule, you can assign classes and ID's to your elements and then select those specific elements

This example targets all elements on the page that have the class exampleClass attached to them and then changes the font size of each to 25px:

.exampleClass {
    font-size: 25px;
}

And here you can see an element that has the ID exampleID attached in which the background colour of the element is being targeted:

#exampleID {
    background-color: 25px;
}

Breakdown and anatomy of a CSS ruleset

There are three aspects that make up a CSS ruleset.

  1. Selector: The element, class or ID of the thing you want to change (p, .exampleClass, #exampleId)
  2. Property: The aspect of the seleected element that you want to change (color, font-size, background-color)
  3. Property Value: The value you wish to assign to the element (red, 20px)

This image from the MDN CSS Basics Page highlights the different sections very clearly:

To add multiple properties to a single element, you simply write them on another line:

img {
    background-color: 25px;
    height: 25px;
    width: 25px;
}

A note on syntax

After assigning the selector, all of the contained properties must be enclosed in curly braces { }

Each property must be ended with a colon :

Each property value must end with a semi colon ;

Selecting multiple elements, classes or ID's

If you want to assign the same style or ruleset to multiple elements, classes or ID's, you can select them by separating each selector with a comma like so:

#exampleID,
.exampleClass,
p {
    background-color: 25px;
}

Various different types of selectors

The examples so far have all been Element selectors (Selecting all HTML elements of that type), ID selectors (Selecting the HTML element with that specific ID type), and Class selectors (Selecting all HTML elements of that type)

You can also use Attribute selectors (which target the elements on the page with that specific attribute) and Pseudo-classs selectors (which target the specified element but only once in a specific state or action is performed on it)

Attribute Selectors:

Syntax:

img[src] {
    border: 1px solid black;
}

Selects the src attribute of an <img> element (<img src=""myImage.png>) but not the <img> element

Pseudo-Class Selectors:

Syntax:

a:hover {
    color: pink;
}

Selects all <a> elements but only when the mouse hovers over it

Fonts and Text

There are a number of built-in fonts available to use with HTML but there are thousands of fonts that can easily be plugged in with a simple <link> tag and Google Fonts

To link a google font and use it, follow these steps:

  1. Navigate to Google Fonts and find something you like
  2. Select a style and weight by clicking the Select XXX ### + button to the right of your selected font
  3. Next, select the icon in the top right to View selected families
  4. Select the <link> radio button then copy the code inside the box
  5. Return to your index.html file and place the copied link into your <head> tag and before the closing </head> tag
  6. Return to the Google Fonts page and now copy the CSS rules
  7. Paste this ruleset into a CSS selector that you wish to use the font family in

Now, you can add this font to any of the CSS selectors for granular use or simply create a ruleset for the <html> tag to apply as default to everything on the page

Boxes or: Everything you see on the internet is actually a box

Layout on a page using CSS is based on the box model with each box (read: element) taking up space on the page

Each box has three properties:

  1. Padding: the space around the content
  2. Border: A solid line that goes around the padding and content
  3. Margin: the space outside of the border

Common CSS Properties

There are almost more properties available in CSS than can feasibly be listed but the most common and likely to be used in just about everything you might do are:

  • width: Sets the width of an element
  • background-color: the colour behind an element and padding
  • color: the colour of an element's content
  • text-shadow: sets a drop shadow on the text inside an element. Useful for providing sharpness to multicoloured text
  • display: sets the display mode of an element. Lot's of variation on this and an entire topic if research in it's own right

Changing the page colour

Using the background-color property alongside the html element, allows you to set the background colour for the entire page

The syntax for the CSS would look like:

html {
    background-color: lightgrey;
}

Styling the body

The body element is no different than any other element on the page and can be styled in much the same way

This page has been styled using the following ruleset:

html {
    width: 600px;
    margin: 0 auto;
    background-color: rgb(109, 102, 102);
    padding: 0 20px 20px 20px;
    border: 5px solid black;
}
