What is HTML?

HTML stands for HyperText Markup Language and is the code used to build and structure web pages. It inlcludes many different types of "tags" that are used to provide semantic meaning and structure to the page.

This includes, but is not limited to:

  • Tables
  • Forms
  • Videos
  • Images
  • Tables
  • Paragraphs
  • Various tpyes of headings

The structure and page is built through a series of "elements" which wrap, encapsulate or enclose the various content on the page as well as styled in various different ways using inline, internal or external CSS. Each of these elements are enlcosed by "tags" that can provide extra functionality (such as an < a > tag to "anchor" the element to an outside link)

Attributes

Attributes provide extra context or information to an element. In the image above, a class name has been added to element. Class is the attribute and "editor-note" is the value that has been assigned to it.

Attributes that set values will always have:

  1. A space between the element name and itself
  2. The attribute that is being calleds name followed by an "=" symbol
  3. The value that is being assigned to the attribute enclosed by speech marks

Nesting Elements

Nesting elements is the practice of enclosing an element wihtin another element. This can be used for various reasons such as anchoring links, turning an image into a buttin or simply making a word more bold or emphasised. Here is an example of how a nested element should be structured:

<p>My pigeon is tiny but <strong>incredibly</strong>vicious.</p>

The <strong> element is nested inside the <p> element and show shows it on the same line

Void

A void element is an element type that has no inner content and no closing tag. The <img> element is a good example of this:

<img src="images/firefox-icon.png" alt="My test image"/>

It contains 2 attributes but does not require a closing tag. This is partly beacuase it's purpose is to enbed the image in the place

Basic Tags and Their Uses

Headings

Headings are used to separate logical chunks of connected text and organise prose. In HTML, heading tags are ordered from 1-6 to show greater or lesser importance. Think of tthe headings as serving similar purposes as:

The <h1> tag is the largest and most important heading. Use it to convey the main title or purpose of the page

The main title of the page (<h1>)

The <h2> tag is then used for things like the introduction in an essay or the first point or section of the page

A top level sub-heading (<h2>)

The <h3> tag is then used for subsections of the <h2> section

A sub-level heading or subsection (<h3>)

The <h4> tag is then used for subsections of the <h3> section

A subsection (<h4>)

The <h5> tag is then used for subsections of the <h4> section

A subsection (<h5>)

The <h6> tag is then used for subsections of the <h5> section

A subsection (<h6>)

Lists

There are two types of list in HTML each of which require two elements to effectively present a list. The two types are "unordered lists" (<ul>) and "ordered lists" (<ol>)

Unordered lists - Shopping lists, notes, objectives. Anything that does not neccesarily need to be done in a specific order

  • Feed pigeon
  • Run away from pigeon
  • Hope pigeon forgets about me

Ordered Lists - Instructions, recipes etc. Anything that requires the list items to be done in order

  1. Open your IDE
  2. Type <ol> and hit return. It should automatically complete the closing tag for you
  3. Inside the new ordered list type <li> to create a list item element. Again, most IDE's will auto complete the closing tag for you
  4. Type the steps between the tags
  5. Congratualtions, you have made an ordered list!

Links

Links are the backbone of the internet. You can link to other pages, other sites, other areas of the same page etc. Adding links is very simple:

  1. Create an anchor tag (<a></a>)
  2. Type the content between the angle brackets where the opening and closing tags meet (<a>Content to be clicked</a>)
  3. Now we need to provide the anchor tag with a href (hyperlink reference) attribute that it will link to when clicked:
    • <a href="https://fullstackopen.com/en/#course-contents">Content to be clicked</a>
